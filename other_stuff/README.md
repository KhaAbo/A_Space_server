# Other Stuff

This folder contains files that are useful for reference or testing but not required for the API to function.

## Contents

### `test_scripts/`

Testing scripts and test outputs:

- `test_api.py` - Python script to test the API
- `test_api.sh` - Bash script to test the API
- `test_output.mp4` - Sample test output

### `sample_videos/`

Old output videos from testing:

- `output/` - Old output videos from testing
- `results/` - Results folder from original gaze estimation

**Note:** Sample input videos are kept in `../gaze-estimation-testing-main/input/` for easy testing with the test scripts.

### `documentation/`

Documentation from the original gaze estimation project:

- `PROJECT_SUMMARY.md` - Technical summary of gaze estimation
- `README.md` - Original gaze estimation README (2 copies merged)

### `old_scripts/`

Scripts from the original gaze estimation project:

- `run_gaze_estimation.py` - Old CLI wrapper (replaced by API)
- `setup.py` - Original setup script
- `requirements.txt` - Original requirements (now in requirements-api.txt)

## Notes

These files are kept for:

- Testing and validation
- Reference documentation
- Sample data for development
- Historical context

They are not needed for the API to run in production.
